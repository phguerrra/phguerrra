## 📌 Projetos em destaque

### 🎟️ EventHubAPI
API REST desenvolvida em **Java com Spring Boot** para gerenciamento de eventos e participantes.  
Possui recursos como login, registro de usuários, CRUD de eventos, inscrição de participantes e controle administrativo.

🔗 Repositório: https://github.com/phguerrra/EventHubAPI

---

### 🛒 StudioZ
Sistema web para personalização de canecas, copos térmicos e garrafas.  
Conta com pré-visualização do produto, cálculo automático de preço, pedidos online e área administrativa.

🔗 Repositório: https://github.com/phguerrra/StudioZ  
🌐 Deploy: https://phguerrra.github.io/StudioZ/

---

### 🎫 ReClick
Aplicação web para reserva e gerenciamento de ingressos de eventos.  
Permite visualizar eventos, realizar reservas e acompanhar informações dos ingressos.

🔗 Repositório: https://github.com/phguerrra/ReClick

---

### 🌱 Sistema de Irrigação
Sistema de irrigação automático utilizando sensor de umidade do solo e microcontrolador, com foco em economia de água e automação.

🔗 Repositório: https://github.com/phguerrra/sistemaIrragacao
